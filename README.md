# 🛒 E-Commerce Sales Dashboard

## 📌 Project Overview
This project presents an interactive **E-Commerce Sales Dashboard** developed for analyzing sales performance and customer purchasing trends. The dashboard provides valuable insights into total sales, order quantity, regional performance, product demand, and customer behavior.

The objective of this project is to help businesses make informed decisions using data visualization and analytics techniques.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify high-performing regions and products
- Understand customer purchasing behavior
- Compare category-wise sales distribution
- Track total orders and quantity sold
- Create an interactive dashboard for business reporting

---

## 📊 Dashboard Features

### KPI Cards
- **Total Sales:** 3M
- **Total Orders:** 100
- **Total Quantity Sold:** 170

### Visualizations Included
- Sales Amount by Region
- Sales Distribution by Gender
- Product-wise Sales Contribution
- Category-wise Sales Comparison
- Customer-wise Sales Analysis
- Interactive Filters (Category, Region, Quarter)

---

## 🛠 Tools & Technologies Used

---

## 📈 Key Insights
- South region generated the highest sales.
- Male customers contributed more to total sales than female customers.
- Laptops and mobiles were among the top-selling products.
- Electronics and Furniture categories showed balanced sales performance.
- Certain customers contributed significantly higher sales amounts.
